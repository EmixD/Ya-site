StatPrism is a statistical software targeting the scientists and students of Psychological and Sociological sciences.

The primary features distinguishing StatPrism:
  • Intuitive Data Analysis — StatPrism simplifies and automates complex data cleaning and analysis tasks.
  • Interactive Plots — StatPrism produces high-quality fully customizeable plots, which can be directly used in scientific articles without further modifications.
  • Analytical, Visual, and Verbal reports — StatPrism provides analytical (tables), visual (plots), and verbal (plain text) forms of data analysis results. The results in StatPrism are automatically converted to the APA format, simplifying and automating the process of writing the article.

Developer Edition

To install the developer edition of StatPrism:

  1. Install Python 3.9.13
  2. Install Git
  3. Clone StatPrism
  4. Run _CREATE_ENV.bat
  5. Run _UPDATE.bat

To update StatPrism: run _UPDATE.bat

To launch StatPrism: run _RUN.bat
